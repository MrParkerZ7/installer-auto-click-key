# Changelog

## v1.0.0 — 2026-06-07

First public release of **AutoKeyClick** (Windows x64, .NET 8 self-contained).

### Artifacts
- `AutoKeyClick-1.0.0-setup.exe` — Inno Setup installer (per-user, Start-menu + optional desktop shortcut, uninstaller, MIT license agreement page).
- `AutoKeyClick-1.0.0-win-x64-portable.zip` — portable build (unzip & run).
- `AutoKeyClick-1.0.0-win-x64.exe` — single-file portable executable (compressed).
- `AutoKeyClick-1.0.0.msix` — MSIX package (self-signed; trust `AutoKeyClick-1.0.0.cer` to install).
- `SHA256SUMS.txt` — checksums for all artifacts.

### Notes
- All builds are self-contained — no .NET runtime install required.
